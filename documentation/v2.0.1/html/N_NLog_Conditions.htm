Contains implementation of conditions used to filter log events.

Classes

  ClassDescription
ConditionEvaluationException
Exception during evaluation of condition expression.
ConditionExpression
Base class for representing nodes in condition expression trees.
ConditionMethodAttribute
Marks class as a log event Condition and assigns a name to it.
ConditionMethods
A bunch of utility methods (mostly predicates) which can be used in condition expressions. Parially inspired by XPath 1.0.
ConditionMethodsAttribute
Marks the class as containing condition methods.
ConditionParseException
Exception during parsing of condition expression.
ConditionParser
Condition parser. Turns a string representation of condition expression into an expression tree.
